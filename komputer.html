CPU (Central Processing Unit)

CPU atau Central Processing Unit adalah bagian terpenting dari sebuah komputer yang bertanggung jawab untuk menjalankan instruksi-instruksi yang diberikan kepada komputer.

Tugas dari CPU yaitu untuk menerima dan menjalankan perintah sebagai perangkat lunak sehingga CPU sering disebut sebagai prosesor dengan kualitas teknologi CPU yang akan digunakan.

Komponen CPU

1.Unit kontrol : Unit kontrol mengelola pemrosesan instruksi dan mengoordinasikan aliran data dalam CPU dan di antara komponen komputer lainnya.

2. ALU (Arithmetic and logic unit) : Melakukan fungsi pemrosesan data komputer.

3. Register : Register adalah lokasi penyimpanan memori kecil berkecepatan tinggi di dalam CPU.

4. CPU interconnection : Beberapa mekanisme yang menyediakan komunikasi di antara unit kontrol, ALU, dan register.

ALU (Arithmetic Logic Unit)

Arithmetic Logic Unit atau yang biasa di sebut ALU merupakan bagian penting dari pada Central Processing Unit atau yang biasa di sebut CPU. ALU pada umumnya memiliki fungsi untuk melakukan perhitungan aritmatika dan logika .

ALU bertanggung jawab untuk melakukan operasi aritmatika serta operasi logika (seperti AND, OR, XOR, dan NOT) dalam sistem komputer.

Peran ALU

1. Operasi Aritmatika: ALU dapat melakukan operasi aritmatika dasar seperti penjumlahan, pengurangan, perkalian, dan pembagian.

2. Operasi Logika:ALU juga dapat melakukan operasi logika seperti AND, OR, XOR, dan NOT. Operasi logika digunakan untuk memanipulasi data biner atau bit-bit data.

3. Perbandingan dan Pemrosesan Data: ALU digunakan untuk membandingkan data dalam CPU.

4. Operasi Bitwise: ALU juga dapat melakukan operasi bitwise pada data. Operasi bitwise memanipulasi bit-bit individu dalam suatu nilai.

5. Penanganan Overflow dan Carry

Register

Register adalah salah satu dari sekumpulan kecil tempat penyimpanan data yang merupakan bagian dari prosesor komputer.

Register dapat menyimpan instruksi, alamat penyimpanan, atau jenis data apa pun (seperti urutan bit atau karakter individual).

Jenis - jenis Register

1. MAR (Memory Address Register): Register ini menyimpan alamat memori dari data dan instruksi.

2. MDR (Memory Data Register): MDR adalah register unit kontrol komputer yang berisi data yang akan disimpan di penyimpanan komputer (misalnya RAM),

3. MBR (Memory Buffer Register): digunakan untuk menyimpan informasi dan data yang dapat dibaca atau ditulis dalam memori komputer.

4. PC (Program Counter): Program Counter digunakan untuk menunjukkan posisi saat ini dari urutan program dalam sistem komputer.

5. Accumulator: Jenis lain dari register CPU yang banyak digunakan untuk menyimpan logika atau hasil sementara.

6. Index Register: Register prosesor yang digunakan untuk memodifikasi alamat operan selama menjalankan program.

Instruction Register: merupakan jenis lain dari register CPU yang digunakan untuk menyimpan instruksi yang sedang dieksekusi atau yang akan didekodekan.
